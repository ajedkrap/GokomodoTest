# GokomodoTest

This repository is meant to fulfill the test for Gokomodo

## Installation

Use the package manager [Node.Js](https://nodejs.org/en/) to GokomodoTest.

```bash
git clone https://github.com/ajedkrap/GokomodoTest.git
yarn install
yarn android
```
